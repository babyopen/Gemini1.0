````
// 总得分 = 基础频次分×动态权重 + 热号惯性加分 + 遗漏回补加分 + 轮转平衡得分 + 多窗口形态加权
得分 + 过热衰减扣分

总得分 = (
  基础分 × 动态权重 + 
  热号惯性加分 + 
  遗漏回补加分 + 
  轮转平衡得分 + 
  形态加权得分 - 
  过热衰减扣分
)
``` 公式构成详解：
1. 基础分 × 动态权重
   
   - 基础分：频次统计得分
   - 动态权重：根据热号惯性、冷号回补等因素动态调整
2. 热号惯性加分
   
   - 利用近期开奖惯性
   - 近期开出的生肖有延续概率
3. 遗漏回补加分
   
   - 长期遗漏的生肖存在概率回补需求
   - 根据遗漏等级（轻冷/深冷/极冷）计算加分
4. 轮转平衡得分
   
   - 12生肖遵循热转冷、冷转热的自然轮转规律
   - 根据生肖当前状态（热/温/冷）计算得分
5. 多窗口形态加权得分 ⭐ V3新增
   
   - 根据7种标准走势给予形态加成
   - 三窗口一致时权重拉满（×1.2）
   - 三窗口全冲突时降低权重（×0.6）
6. 过热衰减扣分 ⭐ V3新增
   
   - 非冷热交替行情时应用
   - 连热超过3期扣40%的热号惯性得分
   - 避免过度追热
````
